# Netflix clone

Netflix 클론은 MVVM 아키텍처를 사용하여 Android에서 생성됩니다. 예고편 재생, 홈 피드, 곧 출시될 영화, 영화 검색/보기, TV 프로그램과 같은 기능이 포함되어 있습니다.
